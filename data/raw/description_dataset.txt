olist_customers_dataset.csv

Набор данных клиентов

Этот набор данных содержит информацию о клиенте и его местонахождении. Используйте его, чтобы идентифицировать уникальных клиентов в наборе данных заказов и найти место доставки заказов.
В нашей системе каждому заказу присваивается уникальный идентификатор customer_id. Это означает, что один и тот же клиент будет получать разные идентификаторы для разных заказов. Цель наличия customer_unique_id в наборе данных — позволить вам идентифицировать клиентов, совершивших повторные покупки в магазине. В противном случае вы обнаружите, что с каждым заказом связан отдельный клиент.

olist_order_items_dataset.csv

Набор данных позиций заказа 

Этот набор данных включает данные о товарах, приобретенных в рамках каждого заказа. 
Пример: 
Order_id = 00143d0f86d6fbd9f9b38ab440ac16f5 содержит 3 товара (один и тот же товар). Для каждого предмета фрахт рассчитывается в соответствии с его размерами и весом. Чтобы получить общую стоимость перевозки для каждого заказа, вам просто нужно просуммировать. 
Общее значение order_item: 21,33 * 3 = 63,99. 
Общая стоимость фрахта равна: 15,10 * 3 = 45,30. 
Общая стоимость заказа (товар + фрахт) составляет: 45,30 + 63,99 = 109,29.

olist_order_reviews_dataset.csv

Набор данных отзывов заказов 

Этот набор данных включает в себя данные об отзывах клиентов. 
После того, как покупатель покупает товар в магазине Olist Store, продавец получает уведомление о необходимости выполнить этот заказ. Как только клиент получает продукт или наступает предполагаемая дата доставки, клиент получает по электронной почте опрос об удовлетворенности, в котором он может оставить примечание об опыте покупки и записать несколько комментариев.
